# Лабораторная работа: Основные задачи DBA в PostgreSQL

### Описание

Студент должен выполнить практическое задание, описанное в файле `task.md`. Задание включает подключение к базе данных PostgreSQL, создание резервной копии базы, восстановление базы, создание индекса и анализ влияния индекса на производительность запросов.

### Файлы

* `task.md` — подробное описание задания для выполнения.
* `lab_postgres_data.sql` — скрипт для автоматического создания таблицы `products` с тестовыми данными.

### Рекомендации по выполнению

1. Ознакомьтесь с инструкциями в `task.md`.
2. Запустите скрипт `lab_postgres_data.sql`, чтобы подготовить тестовую базу данных.
3. Выполните задание шаг за шагом, фиксируя результаты и скриншоты.
4. Составьте отчёт в формате `.doc` или `.docx` с описанием всех шагов, результатами тестов и выводами.

### Результат работы

Студент должен подготовить отчёт в формате `.doc` или `.docx` и предоставить его для проверки. В отчёте необходимо подробно описать каждый шаг выполнения задания, прикрепить скриншоты и сделать выводы о влиянии индекса на производительность запросов.
